# 🤖 Telegram Universal Feedback & Broadcast Bot

![Python](https://img.shields.io/badge/Python-3.9+-blue?style=for-the-badge&logo=python&logoColor=white)
![pyTelegramBotAPI](https://img.shields.io/badge/Library-pyTelegramBotAPI-blue?style=for-the-badge&logo=telegram)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

Ushbu bot ikki xil asosiy funksiyani bajaradi:
1. **Feedback tizimi:** Foydalanuvchilardan murojaatlarni qabul qiladi va admin kanaliga yuboradi. Admin javob qaytarsa, bot uni foydalanuvchiga yetkazadi.
2. **Auto-Broadcast:** Maxsus kanaldan barcha belgilangan guruh yoki kanallarga xabarlarni avtomatik nusxalaydi (copy).

---

## 🔥 Asosiy Imkoniyatlar

- [x] **Ko'p tillilik:** O'zbek va rus tillarini qo'llab-quvvatlash.
- [x] **Ro'yxatdan o'tish:** Telefon raqami va F.I.O ni qabul qilish.
- [x] **Media Support:** Matn, rasm, video, hujjat, ovozli xabar va stikerlarni yubora olish.
- [x] **Admin Reply:** Kanalda foydalanuvchi xabariga "Reply" qilish orqali to'g'ridan-to'g'ri javob yuborish.
- [x] **Smart Broadcasting:** Kanaldagi yangi xabarlarni `TARGET_CHATS` ro'yxatidagi barcha joylarga bir zumda tarqatish.

---

## 🛠 Texnologiyalar

* **Til:** Python 3.9+
* **Kutubxona:** `pyTelegramBotAPI` (Telebot)
* **Xavfsizlik:** `.env` orqali konfiguratsiya (dotenv)
* **Logging:** Xatoliklarni kuzatish uchun o'rnatilgan logging tizimi.
